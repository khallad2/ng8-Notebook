# Angular Notebook

[![N|Solid](appPhoto.png)]()

* Simple app that allows managing of a single notebook with multiple cards
* Possible to create new cards in the notebook
* All cards that are in the notebook are shown on the main screen
* Cards can be moved up and down (order changed) with controls
* Existing cards can be deleted

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.4.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
